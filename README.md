# 🤖 AI Customer Support Crew using CrewAI

This project demonstrates how to build a multi-agent AI customer support system using **CrewAI**.  

It simulates a real-world support workflow where:
1. A **Senior Support Representative Agent** responds to a customer inquiry.
2. A **Quality Assurance Agent** reviews and improves the response.
3. The final polished support answer is generated and saved.

---

## 🚀 Features

- Multi-agent collaboration using CrewAI
- Automated task delegation
- Web documentation scraping tool integration
- Built-in memory for contextual continuity
- Markdown output generation
- Final result saved to a text file

---

## 🧠 How It Works

### 👨‍💼 Agent 1: Senior Support Representative
- Responds to customer inquiries
- Uses website documentation scraping tools
- Provides detailed and complete responses

### 🧑‍🔍 Agent 2: Support Quality Assurance Specialist
- Reviews the drafted response
- Ensures completeness and accuracy
- Improves tone and clarity
- Produces the final polished answer

---

## 🛠️ Tech Stack

- Python 3.11+
- CrewAI
- crewai_tools
- OpenAI (GPT-4o model)
